Title

The title plugin simply searches for the text title and replaces it with the name of the file without the file extension. As an example, below is the default HTML template used by PyMind:

<!DOCTYPE html>
<!-- MARKER -->
<html>
<head>
    <title>title</title>
    <link href="style.css" rel="stylesheet">
</head>
<body>
